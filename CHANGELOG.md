## 2025-01-07 - 添加移动端语言选择器

**提交 ID:** `c8436a8`

### 修改的文件

#### 1. `CLAUDE.md` (新增)
**作用:**
- 项目架构文档，供未来的 Claude Code 实例参考
- 包含代码结构、组件架构、关键模式说明
- 记录常见开发任务和最佳实践

#### 2. `snippets/header-actions.liquid`
**作用:**
- 在移动端头部操作栏中添加语言/国家选择器
- 语言选择器显示在 account button 左侧，仅在移动端显示（< 750px）
- 保持桌面端布局不变

**主要改动:**
- 添加了移动端语言选择器组件（第10-33行）
- 检测是否有多个语言/国家可选
- 使用 `desktop:hidden` 类确保只在移动端显示
- 添加 CSS 样式确保按钮对齐和间距（第228-236行）

#### 3. `snippets/dropdown-localization.liquid`
**作用:**
- 支持移动端显示语言选择器
- 添加 `mobile_only` 参数控制可见性

**主要改动:**
- 添加条件逻辑，根据 `mobile_only` 参数控制是否显示 `mobile:hidden` 类（第24-29行）
- 允许在 header-actions 中显示的同时，保持其他位置的选择器隐藏

#### 4. `snippets/localization-form.liquid`
**作用:**
- 为移动端语言选择器添加专用样式
- 确保在 header-actions 中显示时样式协调

**主要改动:**
- 添加移动端专用 CSS（第820-847行）
- 调整按钮高度、宽度、字体大小以匹配其他 header action 按钮
- 设置下拉面板从右侧对齐
- 调整图标大小和间距

### 功能说明

**移动端（< 750px）:**
- 语言选择按钮出现在头部右侧，位于 account button 左侧
- 点击后弹出下拉面板选择语言/国家
- 原抽屉菜单中的语言选择器保持不变

**桌面端（≥ 750px）:**
- 新添加的移动端语言选择器完全隐藏
- 原有桌面端语言选择器正常显示
- 布局和功能不受影响
